# 로그 이력과 과거 여행
## 여러 커밋과 로그 이력
## 과거로의 시간 여행
### checkout
현재 브랜치에서 과거 커밋 HEAD~로 이동<br>
주요 명령
- $ git checkout HEAD~ : HEAD 이전 커밋으로 이동
- $ git checkout - : 이전 checkout으로 이동
- $ git checkout main : 브랜치의 마지막 커밋으로 이동<br>
현재 상태가 깨끗해야 가능하다.<br>
checkout 후 detached Head 상태로 변경된다. (Head가 커밋ID로 표시됨)
### checkout 되돌아오기
$ git checkout -<br>
$ git checkout main
